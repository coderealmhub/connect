# Connect

Allows you to connect a widget to a third party website using a javascript sdk.

# Embed

    <!-- realm sdk -->
    <div class="up-win" id="realm-root"></div>
    <script type="text/javascript"
    				id="cname"
    				data-cname="{cname}"
    				src="https://connect.realm.com.br/sdk.js">
    </script>
